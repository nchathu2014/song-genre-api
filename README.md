# Song Genre API

## API

## genres
GET | api/genres
GET | api/genres/:id
POST | api/genres
PUT | api/genres/:id
DELETE | api/genres/:id

## customers
GET | api/customers
GET | api/customers/:id
POST | api/customers
PUT | api/customers/:id
DELETE | api/customers/:id

## movies
GET | api/movies
GET | api/movies/:id
POST | api/movies
PUT | api/movies/:id
DELETE | api/movies/:id

## rentals
GET | api/rentals
GET | api/rentals/:id
POST | api/rentals


## Start the project
Install nodemon globally to your host machine
```
npm i -g nodemon

```

Then,

```
nodemon index.js
```
